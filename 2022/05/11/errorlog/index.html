解决

  1. What is *.d.ts[^1]

    • 一文读懂TS的(.d.ts)文件 - 掘金 (juejin.cn)

    • *.d.ts is a type declaration file for JavaScript libraries. Without type declaration file, TypeScript doesn’t work because it doesn’t know the type.

  2. npm i --save-dev @types/marked What does this command do?

    • @types[^2] is a branch of npm to getting *.d.ts files. Before microsoft pulished the TypeScript 2.0, it was little complex to get *.d.ts files, however after TS2.0 was pulished, getting declarations in TS2.0 required no tools apart from[^3] npm. My question is that the beginning of downloading marked doesn’t contain the *.d.ts files, the function of this line of command is to download the files.

  3. Difference between import marked from marked and import {marked} from marked[^4][^5]

    • 进入 index.d.ts 文件里，看到导出的是 function，而没有 default。这个涉及到 import xxx from 和 import {xxx} from 的差别了。只有用 export default 导出的，才能用 import xxx from ，xxx 可以随你自己写，但是没有 export function 意味着人家有规定导出的名字，import {marked} from marked 意思就是从marked 中提取 marked 函数。
